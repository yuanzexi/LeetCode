Given two strings A and B, find the minimum number of times A has to be repeated such that B is a substring of it. If no such solution, return -1.

For example, with A = "abcd" and B = "cdabcdab".

Return 3, because by repeating A three times (â€œabcdabcdabcdâ€), B is a substring of it; and B is not a substring of A repeated two times ("abcdabcd").

Note:
The length of A and B will be between 1 and 10000.

# Solution (python, 48ms, O(m * m) time)
```python
class Solution(object):
    def repeatedStringMatch(self, A, B):
        """
        :type A: str  => target string
        :type B: str  => pattern string
        :rtype: int
        """
        for c in B: # O(m*n) time
            if c not in A:
                return -1
        # determine the range of times
        min_times = len(B)  / len(A)
        
        for times in range(min_times,min_times+3):  # O( m * m) time, where  m = len(B)
            target = A * times
            if B in target:
                return times
        return -1
```
